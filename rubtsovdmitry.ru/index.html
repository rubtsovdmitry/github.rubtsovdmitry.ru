О вёрстке:

Порядок создания проектов — это список проектов представленный ниже с соблюдением хронологии.

Качество вёрстки отличается от ранних до более поздних проектов. Контент для проектов периодически добавляется. Также растёт и качество вёрстки этих добавлений.

Инфраструктура:

Каждый проект находится в своём Docker-контейнере за обратным прокси-сервером Nginx. В Docker-контейнерах используются веб-сервера Apache или Nginx.

Управление инфраструктурой осуществляется по SSH через собственный Wireguard-vpn. Дополнительная защита для SSH – это Fail2ban.

Cтек: HTML & CSS   Контент: Python   Старт проекта: 2023.07.13

Коротко о проекте:

Проект является моим дебютом в IT и посвещён языку программирования Python. Здесь находится множество задач с решениями.

Для создания проекта пришлось познакомиться с инфраструктурой сайтостроения — это система Linux и Web-сервера.

Так как это мой первый проект, то по вопросам верстки здесь всё очень просто — базовые знания HTML и совсем немного CSS.

Основной акцент проекта — это его контент, а точнее язык программирования Python. На этом было сконцентрировано максимум усилий.

Здесь встречаются совсем простые программы и очень сложные с кодом более, чем на тысячу строк. К примеру, игра "Морской бой", в которой было использовано ООП, и рекурсия.

В проекте есть целый раздел, посвящённый сортировке и поиску, рекурсии и другим алгоритмам.

album-new.rubtsovdmitry.ru
Cтек: HTML & CSS + Sass + Django + SQLite3 + s3 object storage   Старт проекта: 2025.07.21

Коротко о проекте:

Тема проекта снова посвящена семейному (фото/видео)альбому.

В проекте впервые мною используется Django-5. Для доставки изображений для CSS и видеоконтента для сайта используется S3 объектное хранилище, но фотоконтент хранится локально в БД SQLite3.

Ссылки на видеоконтент из s3 хранилища генерируются автоматически скриптом, прикрученным к Django. Для добавления нового видеоконтента достаточно его просто поместить в s3 хранилище и видео сразу динамически появляется на сайте.

Для безопасности s3 хранилища ключи от него хранятся в переменных виртуального окружения.

Скиллы / курсы для создания проекта:

  • Современные средства Python разработчика. IDE: PyCharm, VSCode и Spyder [Бауманский учебный центр "Специалист"].
  • Программирование на Python. Уровень 3. Разработка веб - приложений в Django [Бауманский учебный центр "Специалист"].
  • SQL для всех: от начинающих до продвинутых [Stepik].
